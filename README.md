# MHW-Prediccion-Peru
Proyecto Integrador CDA (UTEC). Predicción de la Temperatura Superficial del Mar (SST) con Random Forest Regressor y detección de Olas de Calor Marinas (MHW) en zonas Norte, Centro y Sur del Perú (1990-2025). Incluye pipeline de datos NOAA, evaluación con RMSE/R2 y un Dashboard interactivo.
